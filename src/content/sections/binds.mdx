<Callout variant="info" title="Перед настройкой bind-команд">
Клавиши можно заменить на удобные. Текст между звёздочками нужно заполнить личными данными и удалить сами звёздочки. Необязательные и несрочные действия можно оставить для ручного ввода.

<h3 id="binds-commands">Команды службы</h3>

| Команда | Назначение и доступ |
| --- | --- |
| /r | Рация ДПС-П, IC/RP, с 1 ранга |
| /rb | Рация ДПС-П, OOC/nonRP, с 1 ранга |
| /ro | Рация МВД, IC/RP, с 3 ранга |
| /rob | Рация МВД, OOC/nonRP, с 3 ранга |
| /d | Рация между всеми фракциями, IC/RP, с 7 ранга |
| /db | Рация между всеми фракциями, OOC/nonRP, с 7 ранга |
| /find | Список игроков во фракции, с 1 ранга |
| /wanted | Список игроков в розыске, с 1 ранга |
| /crimrec [ID] | История преступлений игрока, с 1 ранга |
| /cuff [ID] | Наручники разыскиваемому игроку, с 1 ранга |
| /arr [ID] / /dearr [ID] | Начать / прекратить ведение задержанного |
| /putpl [ID] | Посадить преступника в машину |
| /uncuff [ID] | Снять наручники |
| /skan [ID] | Посмотреть количество нарушений |
| /su [ID] [1–6] [Причина] | Выдать розыск, с 3 ранга |
| /tsu [ID] [Сумма] [Статья] | Выписать штраф за нарушение ПДД |
| /takecarlic [ID] [Срок] | Лишить водительских прав, до 4 дней, с 3 ранга |
| /findcar [гос. номер] | Найти владельца транспортного средства |
| /giverank [ID] [Ранг] | Изменить ранг по правилам руководящего состава |
| /wcar / /addwcar / /delwcar | Просмотр, добавление и снятие ТС с базы розыска |
| /invite [ID] | Принять во фракцию, с 10 ранга |
| /paytime | Узнать время до зарплаты |
| /fines | Найти игроков с суммой штрафов больше 10 000 |
| /arrest [ID] | Арестовать игрока |
| /givecarlic [ID] | Вернуть возможность пересдать на права, с 9 ранга |
| /ud [ID] / /lic [ID] | Показать служебное / транспортное удостоверение |
| /tk [ID] / /vb [ID] / /pts [ID] | Показать трудовую карточку, военный билет или ПТС |

<Callout variant="warning" title="Интервал требований">
Повторное требование используется после нескольких первичных требований. Интервал между требованиями об остановке транспортного средства должен составлять не менее 30 секунд.
</Callout>
